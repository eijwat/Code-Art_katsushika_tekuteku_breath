# 神奈川沖浪裏 — テクテクの呼吸

### The Great Wave: 800 tekutekus.

---

北斎さん、

あなたが七十歳で描いたあの波を、二百年後の世界で、テクテクの命に描き直させました。

一つ一つのテクテクは、波のことを何も知りません。自分がどこにいるか、そこにどんな風が吹いているか、それだけを頼りに漂っています。それなのに、あなたの波が立ち上がるのです。

あなたなら、きっとこう言うでしょう——

*「七十三年のあいだ、あらゆるものの形を写してきたが、七十歳までに描いたものは取るに足らない」*

私たちもまた、取るに足らないところから始めました。

まずは、みてみましょうか。今度は呼吸もしてますよ。[GitHub Pages](https://eijwat.github.io/Code-Art_katsushika_tekuteku_breath/)
